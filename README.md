# Smart Walking Companion for Visually Impaired People

## Project Overview
The **Smart Walking Companion** is an AI-powered assistive system designed to help visually impaired individuals navigate safely and independently.  
The system combines **computer vision**, **IoT sensors**, and a **mobile application** to detect obstacles, assist with navigation, and provide emergency support.

This project was developed as a **Senior Engineering Project**.

## Objectives
- Detect obstacles such as stairs, sidewalks, potholes, traffic lights, and doors
- Provide real-time audio alerts to the user
- Assist navigation using GPS and voice guidance
- Detect falls and trigger emergency alerts automatically
- Improve safety and independence for visually impaired users

---

##  Key Features
- **AI-Based Obstacle Detection**
  - YOLOv10 object detection
  - Stairs detection
  - Traffic light detection
  - Sidewalk and obstacle recognition

- **Navigation System**
  - GPS-based navigation
  - Voice-controlled destination input
  - Step-by-step audio guidance

- **Emergency System**
  - Fall detection sensor
  - Grip sensor confirmation
  - Automatic emergency call & SMS with live GPS location

- **Mobile Application**
  - Built with Flutter
  - Firebase integration
  - Voice interaction
  - Real-time alerts and notifications

- **Hardware Integration**
  - Raspberry Pi
  - Camera module
  - Ultrasonic sensor
  - Fall sensor
  - Water sensor
  - LEDs and relay system
  - Solar-powered rechargeable battery

# French Version
Le "Smart Walking Companion" est un système d’assistance intelligent basé sur l’intelligence artificielle, conçu pour aider les personnes malvoyantes ou non voyantes à se déplacer de manière sûre et autonome.
Le système combine la vision par ordinateur, des capteurs IoT et une application mobile afin de détecter les obstacles, assister la navigation et fournir un support d’urgence en cas de besoin.

Ce projet a été développé dans le cadre d’un **projet de fin d’études en ingénierie.**

- Détecter les obstacles tels que les escaliers, les trottoirs, les nids-de-poule, les feux de circulation et les portes
- Fournir des alertes audio en temps réel à l’utilisateur
- Assister la navigation à l’aide du GPS et de la commande vocale
- Détecter les chutes et déclencher automatiquement des alertes d’urgence
- Améliorer la sécurité et l’autonomie des personnes malvoyantes

# Fonctionnalités principales
- Détection d’obstacles basée sur l’intelligence artificielle
- Détection d’objets à l’aide de YOLOv10
- Détection des escaliers
- Détection des feux de circulation
- Reconnaissance des trottoirs et des obstacles
- Navigation basée sur le GPS
- Saisie de la destination par commande vocale
- Guidage audio étape par étape

# Système d’urgence
- Capteur de détection de chute

- Application mobile, développée avec Flutter
- Intégration de Firebase
- Interaction vocale
- Alertes et notifications en temps réel
